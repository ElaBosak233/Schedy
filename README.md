# Schedy

> **课表、今日、上课提醒，零追踪、无埋点。**

---

## 功能概览

- **课表管理**：周 × 星期 × 节次网格，支持多学期、调课、左右滑动切换周
- **今日 / 本周**：快速查看当天或当周安排
- **上课提醒**：基于 `UNUserNotificationCenter` 的本地通知，不依赖任何在线服务
- **小组件**：WidgetKit 读取同一份 SwiftData，桌面/负一屏一眼看到课表
- **导入**：从教务系统（如正方）当前页 HTML 本地解析，或 CSV 导入/导出

---

## 设计原则


| 原则        | 说明                                                                         |
| --------- | -------------------------------------------------------------------------- |
| **纯净**    | 无广告、无拍照搜题、无交友/社区，界面和逻辑只围绕「课程表」                                             |
| **本地优先**  | 课表存 SwiftData，提醒本地调度，导入时仅解析当前页 HTML，不向第三方上传或请求课表数据                         |
| **仅苹果生态** | 基于 SwiftUI、SwiftData、WidgetKit、UserNotifications，面向 iOS / macOS 及配套 Widget |


代码中可直接印证：无广告/统计 SDK、无网络课表 API、无账号与云同步；仅必要的 `URL` 用于打开教务页、设置、关于中的 GitHub 与协议链接。

---

## 技术栈

- **语言 / UI**：Swift、SwiftUI  
- **数据**：SwiftData（本地持久化）  
- **提醒**：UserNotifications（本地调度）  
- **小组件**：WidgetKit
